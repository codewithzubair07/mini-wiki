# Zettelkasten

## Summary
Zettelkasten (German: "slip box") is a personal knowledge management method in which every idea is recorded as a short atomic note and linked explicitly to related notes. Developed and popularised by sociologist [[Niklas Luhmann]], it produces an emergent, non-hierarchical network of knowledge that compounds in value as new notes are added.

## Why It Matters
Zettelkasten provides the theoretical grounding for many modern knowledge tools and, in particular, for the [[LLM Wiki]] architecture. Its four core principles — atomic notes, permanent notes in your own words, explicit links, and no rigid hierarchy — map directly onto the conventions used in this wiki. The method demonstrates that disciplined linking, not volume, is what makes a knowledge base compound over time.

## Key Claims
- Each note must hold exactly one idea ("atomic" principle) (source: [[test-note]])
- Notes are written in the author's own words and kept permanently (source: [[test-note]])
- Every new note references at least one existing note, weaving it into the network (source: [[test-note]])
- Structure emerges from links rather than from folder hierarchies (source: [[test-note]])
- Luhmann's ~90,000-card box supported 70+ books using this approach (source: [[test-note]])
- Digital successors include Obsidian, Roam Research, and Logseq (source: [[test-note]])

## Contradictions / Open Questions
- At very large scale, how does one navigate an unindexed link network without a search layer?
